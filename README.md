# Zenity File Manager

Zenity File Manager is a lightweight, GUI-based file management tool built using **Zenity** and **Bash scripting**. It provides a simple and intuitive interface to perform common file operations such as copy, move, delete, rename, and view file details on Linux systems.

## Features

- Browse directories and files through a GUI
- Copy, move, and delete files or folders
- Rename files or folders
- View file properties (size, type, permissions)
- Easy-to-use, lightweight, and fast
- Runs entirely on Linux with Zenity installed

## Installation

1. Make sure you have **Zenity** installed on your system:
   ```bash
   sudo apt install zenity   # For Debian/Ubuntu-based distros
   sudo dnf install zenity   # For Fedora
   sudo zypper install zenity # For openSUSE
Clone this repository:

git clone https://github.com/pratikmule127/Zenity_File_Manager.git


Navigate to the project directory:

cd Zenity_File_Manager


Make the main script executable:

chmod +x zenity_file_manager.sh


Run the file manager:

./zenity_file_manager.sh

Usage

Launch the script to open a GUI window.

Select a file or folder to perform actions like copy, move, delete, or rename.

Follow the prompts for each action.
